# TO START A LOCAL SERVER
open a terminal instance and run the following command:
```

```

  # TO BUILD AND DEPLOY THE CONTRACT
  open a terminal instance and run the following command:
  ```
  npm run buildD
  ```

  # TO BUILD AND TEST THE CONTRACT
  open a terminal instance and run the following command:
  ```
  npm run buildT
  ```
